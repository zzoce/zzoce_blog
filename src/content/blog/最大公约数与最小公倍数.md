注意负数的情况

**求最大公因数，辗转相除法**：
```Cpp
//此段代码a、b可以为0，且不必区分a与b的大小。
long long int gcd(int a,int b) {
    int t;
    while(b)
    {
        t = a % b;
        a = b;
        b = t;
    }
    return a;
}
```

**最小公倍数lcm：**
```Cpp
long long lcm(long long a,long long b){
    return a*b/gcd(a,b);
}
